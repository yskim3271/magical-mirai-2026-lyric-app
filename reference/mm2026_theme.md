# Magical Mirai 2026 테마 분석 — 「湖のソナーレ」(호수의 소나레)

리릭앱 컨셉에 행사 테마를 녹이기 위한 조사.
단일곡 전용 해석이 아니라, 대상곡 6곡 전체에 적용 가능한 공통 세계관으로 사용한다.

## 1. 테마의 의미

> 「ソナーレ」はイタリア語で「奏でる」という意味。雄大な湖に面した音楽の都・浜松で初開催することから、湖や音楽をモチーフとしたテーマ

- `ソナーレ(Sonare)` = 이탈리아어로 연주하다, 소리내다.
- `湖(호수)` = 하마마쓰의 하마나호. 2026년은 시즈오카현 첫 개최.
- 하마마쓰 = 음악의 도시이자 야마하가 있는 지역. 보컬로이드 기술의 뿌리와도 연결된다.
- 핵심 구도: 호수, 물결, 울림, 반향, 연주.

## 2. 키 비주얼 분위기

- 한여름 한낮의 호수와 맑은 수면.
- 아쿠아, 화이트, 해바라기 옐로, 하늘색 중심의 밝고 청량한 팔레트.
- 수면의 반짝임, 잔잔한 물결, 물과 하늘의 경계가 녹아드는 원경.
- 2025의 밤하늘/은하수 무드와 겹치지 않도록 낮, 물, 소리 쪽을 우선한다.

## 3. 전곡 대응으로 번역할 모티프

| 테마 요소 | 전곡 대응 연출 언어 |
|---|---|
| 소리내다 / 연주하다 | 단어와 비트가 수면, 빛, 입자를 직접 움직인다 |
| 호수 | 배경이 아니라 입력 신호가 반응하는 인터랙션 필드 |
| 반향 / 메아리 | 가사 등장 뒤 잔상, 반복, 반사, 되돌아오는 움직임 |
| 물결 | 단어 타이밍과 보컬 음량이 만드는 파형 |
| 음악의 도시 | 곡별 리듬 구조를 시각적으로 읽을 수 있는 UI |
| 해바라기빛 / 청량감 | 기본 색상과 클라이맥스 발색의 기준 |

## 4. 곡별 차이를 만드는 방식

곡별 고유 가사 해석은 보조로 두고, 아래 데이터 기반 차이를 먼저 만든다.

- BPM/비트 간격: 움직임 속도와 밀도
- 보컬 음량 분포: 폭발력, 크기, 수면 파장
- valence/arousal 중앙값: 색상과 대비
- 코러스 구간: 피크 연출
- 단어 수/문자 수/프레이즈 길이: 가사 배치 밀도
