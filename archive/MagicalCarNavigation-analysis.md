# Magical Car Navigation (2024 대상작) 분석

- **제작**: Team AHEAD (Sarashina, Argynnini) — 3년 연속 참가 끝에 대상, 이듬해(2025)도 우수상
- **소스**: 비공개 (GitHub 공개 안 함) → **공식 배포본의 번들(6MB)을 역분석**
- **데모**: https://magicalmirai.com/2024/procon/entry_2024_app/mm24-procon_entry_app04/
- **지원 환경**: PC 전용 (가로 화면 강제: "画面を横向きにしてください") — PC 전용으로도 대상 수상
- **스택**: React + Leaflet + MapLibre GL(openstreetmap.jp 일본어 베이스맵 스타일) + textalive-app-api

## 컨셉

2024 행사 테마 **「ファンファン・トリップ(FanFun Trip)」을 문자 그대로 직역** — 곡 한 곡 길이만큼,
**실제 오사카(2024 개최 도시!) 지도 위를 자동차로 여행**하는 카 내비게이션 리릭앱.
수상 코멘트: *"여행길에 잔뜩 들르는 寄り道(딴길)을 추억으로 돌아보는 것으로, 여행 전체를 즐기는 리릭앱"*

## 번들 역분석으로 확인된 구현

### 지도·주행 시스템
- **오사카 도로망 GeoJSON을 통째로 번들에 내장** (御堂筋, 国道25号 등 도로명 수백 개 확인) — 라우팅을 클라이언트에서 자체 처리하는 빌드 타임 데이터 파이프라인
- 베이스맵 타일은 런타임에 `tile.openstreetmap.jp` 스타일 로드 — **외부 지도 서비스 런타임 의존이 실전에서 허용된 사례** (단, 핵심 데이터는 자체 번들)
- 자동차는 CSS로 그림: `.car-head-lamp-left/right`, `.car-tail-lamp-left/right` — 헤드/테일램프까지 CSS
- `.historycar` — 과거 주행 흔적 표시로 추정

### 가사 표시
- `.leaflet-lyric-pane`, `.lyric-phrase`, `.tooltip-lyric` — **가사가 지도 레이어(pane)로 렌더**됨.
  주행 중인 길가에 가사가 표지판/말풍선처럼 나타나는 구조

### 게임 메커니즘
- **寄り道(딴길) 시스템**: 지도 위 스팟(食事処, スポーツ施設 등, #筋肉痛 같은 해시태그 포함)을
  클릭하면 들렀다 감 → `want_score × 랜덤`으로 **FanFun도 게이지** 상승
- **곡 구조 = 여정 구조**: 곡마다 `turningPoint1/turningPoint2` 정의 — 곡의 전환점에서 주행 국면 전환.
  findChorus 14회 / findBeat / getValenceArousal / getVocalAmplitude 모두 사용
- **UFO 조우 이스터에그**: `setEncounteredUfo` — 랜덤 인카운터 시 FanFun 보너스 512,810,410점
  (입소문·반복 플레이 장치)
- **엔딩 "旅の思い出(여행의 추억)"**: 곡 종료 후 내가 들렀던 딴길들을 회고 — 매 플레이가 다른 여행이 됨

## StarryLogues와 겹치는 "2년 연속 대상 공식"

| 공식 | 2024 Magical Car Navigation | 2025 StarryLogues |
|---|---|---|
| 행사 테마의 직역 메커니즘 | FanFun Trip → 카 내비 여행 | 星河一天 → 플라네타륨·칠석 |
| **개최지를 콘텐츠로** | 오사카 실제 도로망 | (관측 지점 = 사용자 위치) |
| 곡 구조 = 진행 구조 | turningPoint, 사비=국면 전환 | 사비·비트=연출 전환 |
| 사용자 선택 → 매번 다른 결과 | 어느 딴길에 들를지 | 어떤 단자쿠를 모을지 |
| 엔딩 회고/기념품 | 旅の思い出 화면 | 별하늘 파노라마 PNG |
| 히든 요소 | UFO 조우 | 단자쿠 27개 → 불꽃놀이 |
| 빌드 타임 데이터 파이프라인 | 도로망 GeoJSON | 히파르코스 성표 JSON |

## 2026에 주는 시사점

1. **개최지 직역의 선례**: 2024 대상이 "개최 도시의 실제 지도"를 무대로 썼다 → 2026의 등가물은
   **하마나호(浜名湖)와 음악의 도시 하마마쓰**. 호수 유람·수상 여정·악기의 거리 등이 정확히 같은 수법
2. **"곡 길이 = 여정 길이" 프레임**: 곡을 공간 이동으로 번역하면 전곡 대응이 자연스러워짐
   (곡마다 다른 경로/다른 호수 풍경)
3. **외부 타일 서비스 런타임 의존도 실전 허용** — 단, 핵심 게임 데이터는 자체 번들이 안전
4. **선택→회고 루프**: 사용자가 고른 것들이 엔딩에서 "추억"으로 재구성되는 구조가 2년 연속 먹힘
5. PC 전용 + 가로 강제로도 대상 — 완성도가 플랫폼 커버리지를 이김 (단 2025년은 전 플랫폼이 대상)
